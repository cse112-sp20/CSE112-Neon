# CSE112-Neon

## Installing
```bash
npm i
```

## Running 
```bash
npm start
```


## Commands
	npm start - runs the electron app
	npm lint - runs the linter
	npm test:unit - runs all tests in test/unit_testing
	npm prepush - runs both npm lint and npm test:unit



## Testing
 Unit Testing
